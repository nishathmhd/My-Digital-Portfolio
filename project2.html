Sales Manager WebApp

  • Live Demo
  • Source code

The Sales Manager is a Tkinter-based application designed for efficient data entry and management of hardware sales. It offers a user-friendly interface for inputting customer details, product information, and sales data, all stored in an SQLite database.

Key features include:

  • User-Friendly Interface
  • SQLite Database Integration
  • Efficient Data Entry
  • Modular Code Structure

To run this application, ensure you have the necessary dependencies installed, including Python (3.x recommended), Tkinter library, and SQLite3.

For usage instructions, clone the repository and run the application using the provided commands.
