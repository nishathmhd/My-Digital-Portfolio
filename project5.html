Cloud-Native Monitoring Application Deployment on Kubernetes

This project demonstrates the successful deployment of a cloud-native monitoring application on Kubernetes. It integrates key DevOps tools such as AWS, Python, Docker, and Kubernetes to enhance observability, scalability, and reliability of applications in a real-time environment.

Explore the deployment architecture and configuration, leveraging Kubernetes for container orchestration and management. The application aims to provide comprehensive monitoring and management capabilities for cloud-native environments.
