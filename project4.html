PowerBI Dashboard: Car Supply Chain Management

  • Live Demo
  • Source code

This project showcases interactive PowerBI dashboards for car supply chain management, leveraging a dataset obtained from Kaggle. The dataset underwent exploratory data analysis (EDA), cleaning, and preprocessing using Python before being connected to MySQL for further data manipulation and analysis.

The project features three main dashboards:

  • Order Page
  • Sales Page
  • Customer Review Page

Explore these interactive dashboards to gain insights into the car supply chain management process, including order details, sales performance, and customer feedback.

For access to the PowerBI dashboards and project source code, visit the GitHub repository.
