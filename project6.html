Terraform AWS Infrastructure Automation for Static Website Deployment

  • Live Demo
  • Source code

This project demonstrates the implementation of a DevOps project leveraging Terraform and AWS Cloud to automate the setup of a static website. The project aims to streamline the deployment process, enhance scalability, and ensure efficient infrastructure management.

Explore the infrastructure configuration and automation using Terraform, enabling the deployment of a static website on AWS Cloud with ease. The project highlights the benefits of infrastructure as code (IaC) for maintaining consistent and reproducible deployments.

For access to the live demo and project source code, visit the GitHub repository.
