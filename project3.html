  • London Bike Share

This project features two interactive Tableau dashboards: Project-London Bike Share and Project-British Airways Review.

The London Bike Share dashboard provides insights into bike-sharing usage patterns, including trip duration, start and end stations, and user demographics.

The British Airways Review dashboard analyzes customer reviews and sentiment towards British Airways, highlighting key themes and sentiment trends.

Explore these interactive dashboards to gain valuable insights and visualizations!

For access to the Tableau dashboards and project source code, visit the GitHub repository.
